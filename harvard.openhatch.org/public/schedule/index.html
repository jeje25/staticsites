Why do it?

Open source software like Linux and Firefox are used by hundreds of millions of people. This event is the fast track to participating.

If you come out on Saturday, we'll teach you how the community works. You'll get to play with essential tools like version control and IRC chat.

What are we doing?

In the morning, after coffee and breakfast, we'll have three introductory tutorials:

  • Introduction to Open Source Communication Tools
  • Introduction to the Command Line
  • Introduction to Version Control Using Git

    • We'll share lunch, and then afterwards you'll work with mentors to pick an open source project, find a bug to fix, and make a contribution.

    Coffee will be out at 9am, tutorials start at 10am, and the workshop will go until 5pm.

    Can anyone sign up?

    You have to be a student at Harvard University to attend. (If you want to run a similar event at a different college, email us at events@openhatch.org.)

    We're looking for people who like technical crafts that scale well across the Internet. That usually means software programming, user interface design, technical writing, and testing. And we're looking for people who are enthusiastic!
